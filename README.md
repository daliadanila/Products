# Products
iOS project implemented using MVVM and layered architecture.
Presentation Layer -> Domain Layer <- Data Layer

## Requirements

- Xcode 12
- iOS 13+
- Swift 5

## Concepts used

- MVVM
- Dependency injection
- Data binding using Observable
- Flow Coordinator
